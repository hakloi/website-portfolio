Automated Mounting Scripts for AstraLinux

Description: This project involves creating a series of three Bash scripts for managing mount points on AltLinux. The scripts are designed to automate the following tasks:

  • Mount Point Creation and File Copy: Creates a mount point and copies a specified file from the /tmp directory to the newly created mount point.
  • Disk Space Usage Check: Monitors and reports the disk space usage of the mounted file system using the df command.
  • Mount Point Removal: Unmounts the previously created mount point and removes it.

Technologies: AstraLinux, Bash scripting
