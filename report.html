  4. Accessibility

    5. My portfolio site has been designed with accessibility in mind, incorporating several key practices to ensure it is usable by a broad audience, including those with disabilities:

    • Alt Text for Images: All images on the site include descriptive alt text to ensure they are accessible to screen readers.
    • Text Enlargement Buttons: I added two buttons that allow users to increase the text size on the website. This feature assists users with visual impairments, making the content easier to read and navigate.
    • Color Contrast: High contrast between text and background colors ensures readability for users with visual impairments.
  5. Usability
    • Common CSS File: To avoid the issue of having multiple CSS files for different pages and encountering errors when making changes, I consolidated all CSS into a single common file. This simplifies maintenance and ensures consistency across the site.
    • Navigation Menu: I added a navigation menu to enhance the functionality of the site. This feature allows users to easily navigate between different sections of the site, improving overall usability.
    • Responsive Design: I ensured that the website's layout adapts to different screen sizes by aligning the navigation menu with tablet and desktop versions. However, due to time constraints, the mobile version was not completed.
  6. Learning
    • HTML Forms: I learned how to create an HTML form for a booking system by studying the basics from W3Schools. This was crucial for implementing a feature that allows users to book tickets in advance.
